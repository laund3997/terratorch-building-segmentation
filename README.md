## 🗺️ How It Works

This app fine-tunes pre-trained AI models to focus specifically on buildings in Algiers. It uses satellite images from Sentinel-2, which provides up-to-date and clear views of the Earth’s surface. The AI processes the images to find edges and shapes that match building footprints.

TerraTorch, a deep learning framework, manages this process behind the scenes. This means you do not need to install AI libraries or code anything yourself.
